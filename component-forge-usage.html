Component Forge

How to use the component library

1. Import the generated component

Import the generated dashboard (or other component) from the generated folder:

import { MicroSolveDashboard } from './components/generated/MicroSolveDashboard';
import { Toaster } from 'sonner';

2. Configure theme and container

Set the theme and layout before rendering:

import { Container, Theme } from './settings/types';

let theme: Theme = 'light';   // 'light' or 'dark'
let container: Container = 'none';  // 'centered' or 'none'
Note: Use container: 'centered' only for standalone components, never for full-page apps.

3. Render the component

Use the component in your App and wrap with Toaster for notifications:

function App() {
  const generatedComponent = useMemo(() => {
    return <MicroSolveDashboard />;
  }, []);

  if (container === 'centered') {
    return (
      <>
        <div className="h-full w-full flex flex-col items-center justify-center">
          {generatedComponent}
        </div>
        <Toaster position="top-right" />
      </>
    );
  }

  return (
    <>
      {generatedComponent}
      <Toaster position="top-right" />
    </>
  );
}

4. Theme handling

Apply the selected theme to the document:

function setTheme(theme: Theme) {
  if (theme === 'dark') {
    document.documentElement.classList.add('dark');
  } else {
    document.documentElement.classList.remove('dark');
  }
}
setTheme(theme);

Quick start

  • Run npm install then npm run dev
  • Open the app at the local URL (e.g. http://localhost:5173)
  • Generated components live in src/components/generated/
